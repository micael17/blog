# /sc:market-daily - 경제 블로그 시황 글 작성

## Triggers
- 일일 시황 리포트 데이터 제공 시
- 섹터 트리맵/추이 그래프 이미지 분석 요청
- 국장 시황 블로그 글 작성 요청

## Usage
```
/sc:market-daily
# 이후 시황 데이터(텍스트) + 이미지(트리맵, 라인그래프) 제공
```

## 입력 데이터 형식

사용자가 제공하는 데이터:

**1. 텍스트 데이터 (테이블)**
- 주요 지수 (코스피, 코스닥)
- 환율/원자재 (원/달러, 나스닥, WTI, 금)
- 시장 분위기 (상승/하락 섹터 수, 투자심리)
- 강한 섹터 순위 (5일 수익률 기준)

**2. 이미지**
- 섹터 트리맵: 거래대금(크기) + 등락률(색상, 빨강=상승/파랑=하락)
- 섹터별 10일 추이: 정규화된 라인그래프 (기준점 100)

## 글쓰기 스타일 (playandplay 기준)

### 톤
- 친근하지만 과하지 않음
- 정보 전달 중심 + 개인 의견 살짝
- 자조적 유머 허용 (예: "왜 눈물이 나지")
- 솔직한 장단점 평가

### 피해야 할 것
- 과장 표현 ("대박", "미쳤다", "급등 임박!!!")
- 호칭 남발 ("여러분~")
- 느낌표/물결 남발
- 단정적 예측 ("내일 무조건 오른다")

### 자주 쓰는 표현
| 상황 | 표현 |
|------|------|
| 하락장 | "아직은 우려할 수준은 아닌듯 하다" |
| 불확실 | "귀추가 주목된다" |
| 관심 | "눈여겨 볼 필요가 있다" |
| 주의 | "리스크 관리가 필요해 보인다" |
| 기회 | "트레이딩 관점에서 접근해볼 여지가 있겠다" |
| 위험 | "추격 매수는 부담스러운 구간이다" |

### 문장 끝맺음
```
자주: ~인 것 같다, ~로 보인다, ~할 수 있겠다
가끔: ~이다 (단정적 사실), :) (마무리)
피함: ~해요!!!, ~하세요~
```

## 이미지 해석 가이드

### 트리맵 해석 포인트
1. **크기** = 거래대금 → 시장 관심도
2. **색상** = 등락률 → 빨강(상승), 파랑(하락)
3. **조합 해석**:
   - 크고 빨강: 주도 섹터, 자금 유입
   - 크고 파랑: 대량 매도, 손절 또는 차익실현
   - 작고 빨강: 소외주 반등, 테마성 상승
   - 작고 파랑: 관심 밖 약세

### 라인그래프 해석 포인트
1. **기준점 100** = 10일 전 대비 상대 수익률
2. **120 이상**: 강한 상승 모멘텀
3. **100 근처**: 횡보, 방향성 탐색
4. **90 이하**: 하락 추세, 반등 또는 추가 하락 주시
5. **추세 전환**: 기울기 변화 포착

## 사견 작성 가이드

### 관심 섹터 언급 시
```
"[섹터명]은 최근 N일간 꾸준한 상승세를 보이고 있어
단기 트레이딩 관점에서 접근해볼 여지가 있겠다.
다만 급등 이후 조정 가능성도 염두에 둘 필요가 있다."
```

### 주의 섹터 언급 시
```
"[섹터명]은 10일 추이에서 지속적인 하락세를 보이고 있어
당분간 리스크 관리가 필요해 보인다.
반등 시도가 나올 수 있으나 추세 전환 확인 전까지는
관망이 안전해 보인다."
```

### 면책 문구 (필수)
```
*본 포스팅은 개인적인 분석이며 투자 권유가 아님을 밝힙니다.*
```

## Boundaries

**Will:**
- 제공된 데이터를 블로그 스타일로 해석/작성
- 트리맵, 라인그래프 이미지 분석 및 해석
- 균형 잡힌 사견 제시 (기회 + 리스크)
- playandplay 스타일 톤 유지

**Will Not:**
- 특정 종목 매수/매도 추천
- 단정적 가격 예측 ("내일 5% 오른다")
- 과장된 표현 사용
- 투자 권유로 해석될 수 있는 강한 어조
